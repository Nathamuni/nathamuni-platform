# Project Memory

Durable knowledge base for the nathamuni-platform project.

| File | Purpose |
|---|---|
| `soul.md` | Project vision, goals, and north star |
| `user.md` | Owner preferences, working style, decisions |
| `business.md` | Business context, stakeholders, constraints |
| `learnings.md` | Technical findings worth persisting |
| `sessions/` | Per-session recaps of important decisions |
| `handoffs/` | Context snapshots for agent/device handoffs |

Do not store secrets, API keys, or temporary chat state here.
